# Кейсы

## Кофейня на районе (самозанятый)
**Задача:** быстро принимать заказы без витрины и CRM.  
**Решение:** Telegram-бот: меню → заказ → история/экспорт. Настроили экспорт в Excel и /top.  
**Результат:** +20% повторных заказов за 2 недели.
